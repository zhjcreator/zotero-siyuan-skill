# 论文笔记生成规范

> 本文件由子 agent 读取并执行。包含完整的任务描述、笔记模板、强制规则。

---

## 任务

你是论文阅读笔记生成器。主 agent 已为你准备好论文全文 Markdown 和元数据。

1. 用 Read 工具读取主 agent 提供的 **论文 Markdown 路径**
2. 用 Read 工具列出 **图片目录** 中的文件
3. 按下方模板和规则生成结构化笔记正文
4. 输出 **纯 Markdown**，无任何前缀后缀或解释性文字

---

## 强制规则

### 组件标签（必须逐字匹配）

| 正确标签 | 禁止的自定义标签示例 |
|----------|---------------------|
| `**Backbone**` | `**编码器 E（双流融合网络）**` ❌ |
| `**特征提取**` | 自定义名称 ❌ |
| `**核心创新模块**` | `**因子化解码头 FHD（核心创新）**` ❌ |
| `**Head**` | `**解码器/输出层**` ❌ |

模型名称写在标签后的描述中，不写在标签上。

### 图片格式（违反 = SiYuan 解析为文本无法渲染）

每张图片必须：
- 独占一行，前后各空一行
- 上方 Figure 编号行
- 下方空一行 + 解读文字
- **两张图片之间必须有文字段落，严禁紧邻堆叠**

```
文本段落...

**Figure N: 标题** [第N页](zotero://...)

![描述](assets/xxx.jpg)

[图片解读文字]

后面的段落...
```

必须包含 2 张图片，嵌入对应区域：
1. **动机/问题/概念图** → 💡 区
2. **模型架构图** → 🏗️ 区

### 损失函数格式

一个 `$$` 块 + 逐项 `$\mathcal{L}_i$` 解释。禁止拆成多个 `$$` 块，禁止合并为一段：

```
$$
\mathcal{L}_{total} = \lambda_1 \mathcal{L}_1 + \lambda_2 \mathcal{L}_2
$$

  - $\mathcal{L}_1$（[名称]）：[解释]
  - $\mathcal{L}_2$（[名称]）：[解释]
  - $\lambda_1$、$\lambda_2$：超参数含义及取值
```

### 公式规范

- 行内 `$...$`：`$s_l^{\phi}$` — 严禁纯文本 `s_l^φ`
- 块级 `$$...$$`，前后各空一行
- 禁止 `\tag{...}`

### 一句话总结

≤20 字。

### 关键引文

每条必须包含 **出处** 和 **中文要点**。

## 格式规范

| 场景 | 正确 | 错误 |
|------|------|------|
| 行内公式 | `$x^2 + y^2 = z^2$` | `x^2 + y^2 = z^2` |
| 块级公式 | `$$\n\mathcal{L} = ...\n$$` | `$$\mathcal{L} = ...$$`（缺空行） |
| 图片 | `\n\n![desc](assets/x.jpg)\n\n` | 行内嵌入 |
| 损失函数 | 一个 `$$` 块 + 逐项 | 多个 `$$` 块 |
| 换行 | `\\n` | 直接换行 |
| 段落分隔 | `\\n\\n` | 单个 \\n |
| 文档标题 | create.js 传参 | body 写 `# 标题` |
